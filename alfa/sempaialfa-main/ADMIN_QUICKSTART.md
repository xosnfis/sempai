# Быстрый старт админ-панели

## Шаг 1: Создание суперпользователя

```bash
cd sempaialfa-main/aichat
python manage.py createsuperuser
```

Введите:
- Username (имя пользователя)
- Email (опционально)
- Password (пароль)

## Шаг 2: Запуск сервера

```bash
python manage.py runserver
```

## Шаг 3: Вход в админ-панель

1. Откройте в браузере: `http://localhost:8000/admin/`
2. Введите учетные данные суперпользователя
3. Нажмите "Войти"

## Основные разделы

### Запросы к AI (`/admin/main/chatrequest/`)
- Просмотр всех запросов
- Фильтрация по статусу
- Детальный просмотр запроса и ответа
- Просмотр метрик запроса

### История чатов (`/admin/main/chathistory/`)
- Все чаты пользователей
- Просмотр сообщений
- Статистика по чатам

### Метрики (`/admin/main/metric/`)
- Агрегированные метрики качества
- Отслеживание целей
- Фильтрация по категориям

### Метрики запросов (`/admin/main/chatrequestmetrics/`)
- Детальные метрики каждого запроса
- Производительность
- Качество ответов

## Полезные функции

### Поиск
Используйте поле поиска вверху списка для быстрого поиска по:
- ID запросов
- Тексту сообщений
- Email пользователей
- Названиям метрик

### Фильтры
Используйте боковую панель фильтров для:
- Фильтрации по датам
- Фильтрации по статусам
- Фильтрации по категориям

### Детальный просмотр
Нажмите на любой объект в списке для просмотра всех деталей:
- Полный текст запроса/ответа
- JSON данные
- Метрики
- Статистика

## Горячие клавиши

- `Ctrl+F` - Поиск на странице
- `Tab` - Переключение между полями
- `Enter` - Сохранение формы

## Советы

1. **Используйте фильтры** - они помогают быстро найти нужные данные
2. **Проверяйте метрики** - они показывают качество работы системы
3. **Смотрите ошибки** - в ChatRequest есть поле "error" для просмотра ошибок
4. **Используйте inline метрики** - в детальном просмотре запроса метрики отображаются прямо в форме

## Решение проблем

### Не могу войти
- Проверьте, что создали суперпользователя
- Убедитесь, что используете правильные учетные данные

### Не вижу данные
- Убедитесь, что есть данные в базе
- Проверьте фильтры - возможно, они скрывают данные

### Медленная загрузка
- Используйте фильтры для уменьшения количества записей
- Проверьте размер базы данных
